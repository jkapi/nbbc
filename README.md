# NBBC - BBCode to (X)HTML Parser

**A high-speed, extensible, easy-to-use validating BBCode parser that accepts BBCode as input and generates XHTML.**

**Requires:** PHP 4.0.5+, or PHP 5.0+, or PHP 7.0+
